How it works

From pipe to dashboard, in three steps.

01 / SENSE

Low-cost sensors

Acoustic and pressure sensors clip onto existing pipe infrastructure at key network points, no major excavation needed.

→
02 / ANALYZE

AI-driven detection

Machine learning models flag pressure anomalies and acoustic signatures consistent with leaks, before they surface.

→
03 / ACT

Prioritized alerts

Utility crews get a ranked list of likely leak locations, so repairs target the biggest losses first.

Main features

Built for the control room, not the boardroom.

Every feature is designed around what a utility operator actually needs to act on, fast.

Live network map

A real-time view of pressure and flow across the distribution network, with leak probability overlaid by zone.

Prioritized repair queue

Alerts ranked by estimated water loss and confidence, so crews always know where to go first.

Historical loss reporting

Track non-revenue water trends over time, useful for budget requests and investment justification.

Works with existing sensors

Compatible with low-cost, locally available hardware, no proprietary sensor ecosystem required.
